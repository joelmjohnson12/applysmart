# ApplySmart

ApplySmart is a Rust CLI tool that automates generating tailored CVs and cover letters for job applications.  

## How it works
- Input your info in `example.yaml`
- Templates are in `templates/`
- Run the tool (or push files to GitHub)
- Output is in `output/` folder
